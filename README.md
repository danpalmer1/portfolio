| Layer | Choice |
|---|---|

| Fonts | Self-hosted Geist Sans + Geist Mono (`@fontsource-variable`) |
| Scroll | Lenis (smooth) bridged to GSAP/ScrollTrigger (choreography) |
| Background | React Three Fiber + capability-gated CSS fallback |
| Content | MDX Content Collections (Zod-validated) + Shiki `github-dark-default` |
| SEO | `@astrojs/sitemap` + `@astrojs/rss` + reusable `<SEO/>` component |
| Hosting | AWS Amplify (planned вЂ” Phase 3) |
| Contact backend | SES + Lambda + API Gateway + Cloudflare Turnstile (planned вЂ” Phase 3) |

## Design system вЂ” Midnight Tide

Dark-only v1 (`color-scheme: dark`). All palette tokens live in `@theme` inside `src/styles/global.css`:

- **Backgrounds:** `--color-bg-deep` `--color-bg` `--color-surface` `--color-surface-2` `--color-border`
- **Blues:** `--color-accent-deep` `--color-accent` `--color-accent-glow`
- **Text:** `--color-text` `--color-text-muted` `--color-text-dim`
- **Type:** Geist Sans display + body, Geist Mono for monospace. Display weight 600, tracking -0.03em.

Every text/background pair passes WCAG AA. See `scripts/contrast-check.mjs` to verify.

## Performance + accessibility

- **Static-first.** Every route is pre-rendered at build (`output: "static"`).
- **Islands only where needed.** Two: `<BackgroundFXClient client:only="react"/>` (capability-gated) and `<ContactForm client:load/>`.
- **Perf budget:** landing-route critical JS в‰€ 112 KB gzipped (under 200 KB target).
- **Progressive enhancement:** every page renders without JS вЂ” the WebGL scene and scroll choreography layer on top.
- **`prefers-reduced-motion`** honored across Lenis, GSAP, and the WebGL gate.
